INSTRUKSI PENGGANTIAN SCREENSHOT APLIKASI MOBILE

Untuk mengganti screenshot aplikasi mobile di website, ikuti langkah-langkah berikut:

1. Persiapkan 2 screenshot aplikasi mobile Anda dalam format PNG:
   - Pastikan ukuran gambar ideal sekitar 1080px x 2340px (atau rasio sekitar 9:19)
   - Pastikan kualitas gambar baik dan jelas

2. Ganti file-file berikut dengan screenshot aplikasi Anda:
   - app-screen-1.png (untuk tampilan "Scan Keamanan")
   - app-screen-2.png (untuk tampilan "Analisis Risiko")

3. Pastikan penamaan file tetap sama persis seperti aslinya

4. Untuk hasil terbaik, gunakan screenshot dengan latar belakang yang kontras dan tampilan UI yang jelas

5. Restart aplikasi setelah mengganti file-file tersebut untuk melihat perubahan

Catatan: Jika Anda ingin mengganti judul atau deskripsi untuk setiap screenshot, Anda dapat mengedit file src/components/Showcase.tsx bagian appScreenshots array.
